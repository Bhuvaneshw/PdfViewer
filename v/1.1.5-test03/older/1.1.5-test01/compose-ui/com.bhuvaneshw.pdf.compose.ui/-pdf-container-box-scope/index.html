PdfContainerBoxScope

class PdfContainerBoxScope : PdfContainerScope, BoxScope

A scope for composables within a PDF viewer container that is also a BoxScope. Provides access to both the PDF viewer state and the receiver scope of a androidx.compose.foundation.layout.Box composable.
