PdfScrollBar

class PdfScrollBar @JvmOverloads constructor(context: Context, attrs: AttributeSet? = null, defStyleAttr: Int = 0) : FrameLayout

A scrollbar for the PDF viewer that can be dragged to navigate through pages. It can be either vertical or horizontal, and shows the current page number.
