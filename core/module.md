# Module core

This module contains the core `PdfViewer` view, which is a highly configurable PDF viewer built on top of `WebView` and Mozilla's `PDF.js`. It provides a comprehensive set of features for PDF viewing and interaction, including:

*   Loading PDFs from various sources (assets, files, content URIs, and URLs).
*   Navigating through pages, zooming, and searching for text.
*   Editing capabilities like highlighting.
*   Customizable UI and behavior.
